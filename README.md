## Pinball Prototype

Open `BallGame` with Unity 2022.3.62f3, then open `Assets/Scenes/PinballScene.unity` and press Play. The scene installer creates a visible neon-style 2D pinball table and connects the imported feedback sounds.

Controls:

- `Space`: hold to charge, release to launch.
- `A` / `LeftArrow`: left flipper.
- `D` / `RightArrow`: right flipper.

Third-party reference notice: `BallGame/Assets/Pinball/Docs/ThirdPartyNotices.md`.
